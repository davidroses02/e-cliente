DNI

  1. Abre la consola para ver también el resultado
  2. En este ejercicio hay comunicación con el árbol DOM:
    • document.getElementById()
      • obtiene el elemento del árbol DOM
    • .addEventListener("click", callback)
      • asocia un evento a un elemento del árbol DOM
    • mensajeSpan.textContent:
      • actualiza el contenido del span
    • document.addEventListener("DOMContentLoaded", callback)
      • imprescindible para poder trabajar con los elementos del árbol DOM
